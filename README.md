# Git for project management

This is an example Git repository that is used as part of the Git for project management workshop.

The aim of this course is to demonstrate how Github can be used to manage your project and collaborate with others on a project effectively.

For the purposes of this course we will consider building a bungalow with the following plan:

1 - 🌍 Site Survey & Preparation: Conducting a site survey, clearing vegetation, levelling the plot, and setting out the building footprint.

2 - ⛏️ Foundations: Excavating trenches, pouring concrete, installing drainage, and constructing the damp-proof course (DPC).

3 - 🧱 Superstructure: Building the external and internal load-bearing walls, installing lintels, and constructing the roof structure with tiles or felt.

4 - 🪟 Make Weathertight: Fitting external windows and doors, sealing all openings, and ensuring the building is protected from the elements.

5 - 🔧 First Fix: Installing internal stud partition walls, running plumbing pipes, electrical wiring, and ducting for heating systems before plastering begins.

6 - 🪣 Plastering & Screeding: Covering interior walls and ceilings with plaster, and laying floor screed to create a smooth, level base.

7 - 🔌 Second Fix: Installing finished electrical items (sockets, switches, and consumer unit), plumbing fixtures (taps, boiler, and radiators), internal doors, and kitchen and bathroom fittings.

8 - 🎨 Final Finishes & Snagging: Decorating walls and ceilings, laying flooring, completing external landscaping, and carrying out a snagging inspection to identify and resolve any outstanding defects.

Please try and add git issues and sub issues based on this plan to this repository. I'll take you through an example first.
